1.熟悉键盘及其键位

1.1：全尺寸键盘

目前我们使用的台式机键盘基本都是全尺寸键盘，什么叫全尺寸的呢？如下图所示，就是键盘有的按键，这里都有。学校的机房里面的键盘，基本都是全尺寸的键盘。

1.2：全尺寸键盘的键位区分 

1
    全尺寸键盘有那么多的按键，每个按键都有什么功能呢？接下来，我们来认识一下。
  • 功能键区。功能键区的按键，通常都是一些快捷键的设置，比如调高音量、调高亮度、关闭程序等。
  • 主键位区。主键位区包括了我们需要用到的绝大部分按键，比如输入大小写字母、数字、符号等。
  • 辅助键区。辅助键区主要是辅助功能，比如上下左右的按键，光标定位到首行（Home键）等。
  • 小键盘区。小键盘区是数字区域，这个区域是主要方便我们进行数字计算，包含了数字键，加减乘除键，类似我们使用的计算器一样。
  • 指示灯区。指示灯区是提示区，比如我们开启大写输入、开启小键盘区之后，指示灯区就会对应的亮起来。

1.3：常见的按键功能

刚才我们一起熟悉了键盘键位，好学的小梦就开始琢磨起这些按键的具体用处了，小梦问小马：“小马小马，刚才我们熟悉了键盘键位，但是你知道这么按键的具体作用吗？”小马开心的说到：“哈哈，我知道呀，正好之前做好了功课，我们一起来看看吧！”

  • 退出键：主要是在全屏状态下的退出操作，比如老师播放课件的时候，是全屏显示的，如果要写代码了怎么办呢？可以按一下ESC键，就可以退出全屏模式了。
  • 制表符：制表符在我们后续的学习中会经常用到，主要是起到缩进的作用，相当于4个空格键。
  • 大写锁定键：如果我们要输入大写字母，可以按一下这个按键，此时指示灯区会有灯亮起，我们就可以输入大写了。要切回小写，再次按一下，指示灯灭了就可以再次输入小写字母。
  • 上档键：如果我们要输入“#”，该怎么输入呢？可以看到，#在主键位区的数字“3”这个按键，按一下会出来数字3，而不是“#”，这个时候就需要借助上档键了，按住上档键不要放，然后再按“#”，你会发现，“#”神奇的出现了。有的按键是有上下两种不一样的符号的，如果要输入上面的符号，就需要借助上档键完成操作。
  • 组合功能键：顾名思义，就是跟其他按键的组合作用，下一节中我们会有介绍。
  • Win键：主要是弹出计算机的开始菜单，有利于我们打开其他的程序。
  • 交替换挡键：跟其他的按键进行组合使用，下一节中我们会有介绍。
  • 退格键：当我们输入了错误的信息的时候，我们要删除，按这个按钮就可以把光标之前的信息删除。
  • 回车键：有换行、确定的功能。
  • 插入键：当你发现你的光标变粗了以后，你可能就是按到了这个键，主要作用是把光标定位的信息进行替换，如果你按到了这个键，再按一次就可以回到正常的操作。
  • 删除键：与退格键相反，这个按键可以把光标后面的信息进行删除。

2.2：标准打字法

这些单词组合在一起是什么意思呢？好学的小梦戳了戳小马，小马也表示疑问，于是请教老师，老师说：“这些看上去专业的让人看不懂，其实跟我们的生活倒还是蛮贴切的，我们一起来看看！”。

#include<bits/stdc++.h> :首先是 # ，这个是固定的语法，就好比我们写作文的时候，每段的开头都要空两格。后面的 include 是“引入”的意思

using namespace std; ：这句话的意思是“使用标准的命名空间”，什么意思呢？这句话大家不需要去纠结什么意思，大家就当作一个语法来用就好了，比如我们写信的时候，最后写上“此致敬礼”。

int main() ：这是程序的主函数。打个比方，我们去学校学习，想要进入教室、机房、礼堂和食堂等，首先只能从学校的正大门进去，其他地方不能进去。同样的，我们的程序执行也是从这一句开始的。可以把 () 看作学校的大门，而后面跟着的 {} 就是学校里面的围墙。记住：括号都是成双成对的哦！

return 0; ：返回一个0。为什么要返回一个0呢？又是返回给谁呢？其实呀，0是代表结束的意思，到了这一句，就代表外面的程序结束了。就像我们听到了当天最后一节课的下课铃声，我们就知道了可以放学回家了，最后一节课的下课铃声就是一个标志，标志着放学回家了。正大门的保安叔叔听到了也知道可以放同学们出去回家了。这里的0也是返回给主函数的。

  • 几点要注意的：

    • 输入法要在英文状态下输入；
    • 基本上每一行除了以括号结尾，其他的都是以 ; 结尾哦，就像我们以 。 结束一句话一样；
    • 大括号里面的内容都要缩进一个 Tab 键哦，相当于四个空格。

写好的代码，我们是需要运行起来的，这样才可以看到结果。我们可以在桌面上新建一个文件夹，命名为C++ source file 。在运行之前，我们需要保存我们的代码，直接使用快捷键 Ctrl+S ，就会弹出保存文件的框，选择我们刚才创建的文件夹，然后把我们写好的代码命名为wawawa.cpp，点击确定即可报错。

注意：文件名可以随意选取，命名一定使用英文命名。

 

 

3.4：文件的编译和运行

激动人心的时候到了！编辑并保存代码之后，如何知道我们写的代码是否正确呢？强大的编译运行功能来了。

如图所示图标：1编译、 2运行、 3编译运行 。一般我们直接点击图标3，就能完成项目的编译运行。

 

那么编译和运行究竟是怎么回事呢？

  • 编译：编译其实就相当于我们的英语翻译一样。当我们写好了程序之后，这个程序只是我们看得懂的程序，并不是计算机可以懂的程序，我们就要使用这个功能，把我们的程序“翻译”成计算机可以理解的程序。如果有错误，那么编译完了之后，程序中会有光标定位，并且定位的这一行是红色的，初学者的错误，一般是上一行是否忘记了以 ; 结尾，或者是当前行和上一行使用了中文的分号。
  • 运行：程序无误之后，就可以运行啦。瞧，弹出了一个黑色的框框。里面显示了 “Wa Wa Wa!”的字样。好啦，这就是我们写的第一个程序，发出的第一句问候。

4.1：五则运算

我们知道，数学上有 +、-、*、/ 四则运算（乘法用 * 代替，除法用 / 代替），C++语言中，多了一种运算，叫“模运算”。首先，C++的加法、减法和乘法运算与数学中的运算法则一致，如上图的例子。唯一区别的就是除法和模运算。

  • 除法：当除数和被除数都是整数的时候，出来的结果就是整数，比如 3/2=1，同学们就会奇怪了，不应该是1.5 吗？这个就是区别的地方了，如果要出来小数呢？我们把规则研读一遍：当除数和被除数都是整数的时候结果就是整数，那么我们破坏其中的一个条件呢？bingo！只要其中一个不是整数，结果就出来小数了，比如 3.0/2=1.5 ，这样就处理好了。

  • 模运算：符号是 %，代表模运算，实现的功能是：对一个数取余数！神奇不？我们要对一个数取余数，本来是要先算出商，再求余数的，这里直接就可以取余数了，作用和加、减、乘、除是一样的，比如 7%2=1 。
